# Apprendre-le-Russe

Voici une page simple html avec une base de données de mots très utilisé en français et en russe afin de pouvoir s'entraîner.

L'objectif est simple, des mots russes s'affiche et il faut les écrire en phonétique latine ou en traduction française. 
Une note sur 20 est donnée à chaque réponse et la moyen est calculé.

Futur :
- La prochaine étape sera de rajouter un bouton audio qui lie le mot russe demandé.
